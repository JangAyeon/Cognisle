## Cognisle
**기억력 테스트를 통한 섬 꾸미기 웹 어플리케이션**

## 사이트 주소
https://cognisle.vercel.app

## 주요 기능
1. 기억력 테스트 (4x4 카드 뒤집기 게임)을 통해 아이템 획득
2. 보유한 아이템으로 나의 섬 꾸미기
3. 친구의 섬 놀러가기
4. 보유한 아이템에 한해서 나와 친구들의 공부 데이터 분석 데이터 담긴 아이템 카드 열람 가능
